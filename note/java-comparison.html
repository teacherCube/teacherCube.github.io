Comparison

Comparison operators are used to find the relationship between two values. They are also known as relational operators.

NameOperatorExample
More than>a > b
Less than<a < b
More than or Equal to>=a >= b
Less than or Equal to<=a <= b
Equal to==a == b
Not Equal to!=a != b

More than (>) operator returns true if value of a is bigger than value of b, otherwise it returns false.

int a = 5; 
int b = 8; 
System.out.println(a > b);// displays false

Less than (<) operator returns true if value of a is smaller than value of b, otherwise it returns false.

int a = 5; 
int b = 8; 
System.out.println(a > b);// displays true

Equal to (==) operator returns true if value of a is equal to value of b, otherwise it returns false.

int a = 5; 
int b = 8; 
System.out.println(a == b);// displays false
System.out.println(b == 8);// displays true

More than or equal to (>=) operator returns true if value of a is bigger than value of b or if the values are equal, otherwise it returns false.

int a = 5; 
int b = 8; 
System.out.println(a >= b);// displays false
System.out.println(a >= 5);// displays true

Less than or equal to (<=) operator returns true if value of a is smaller than value of b or if the values are equal, otherwise it returns false.

int a = 5; 
int b = 8; 
System.out.println(a <= b);// displays true
System.out.println(a <= 5);// displays true

Not equal to (!=) operator returns true if value of a is not equal to value of b, otherwise it returns false.

int a = 5; 
int b = 8; 
System.out.println(a != b);// displays true
System.out.println(b != 8);// displays false

Logical

NameOperatorExample
Logical And&&a && b
Logical Or||a || b
Logical Not!!a

Logical And (&&) operator returns true if value of a and value of b is true, otherwise it returns false.

int a = 5; 
int b = 8; 
System.out.println(true && true);// displays true
System.out.println((a < b) && (a < 10));// displays true
System.out.println((a > b) && (a < 10));// displays false

Logical Or (||) operator returns true if either one or both value of a and value of b is true, otherwise it returns false.

int a = 5; 
int b = 8; 
System.out.println(true || false);// displays true
System.out.println((a == 1) || (b < 5));// displays false
System.out.println((a < b) || (b < 10));// displays true

Not(!) operator returns true if value of a is false, and returns false if value of a is true.

int a = 5; 
int b = 8; 
System.out.println(!(a < b));// displays false
System.out.println(!(3 < 1));// displays true
