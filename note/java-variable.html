Variable

Variable is a placeholder that store a data in a program. The value of data can change during program execution.

Variable declaration is creating a variable in a program to be used for different purposes in that program. Variable is declared with following syntax:

dataType variableName;

The dataType are keyword such as int, char or double. The variable names are identifier in example are length, grade, and distance.

For example, declare a integer variable to count cars that uses a road.

int countCars;

By convention, variable names are written using lowercase letter, cannot have space and do not start with digits. For example,

  • height
  • shortestDistance
  • grossProfitMargin

Declaring multiple variables of same data type

dataType variable1, variable2;

For example, width, height and length are double: 

double width, height, length;

Variables initialization is assigning an initial value of the variable.

dataType variableName = value;

for example

int a = 0;

Data types

Java supports data type such as byte, short, int, long, float, double, char, and Boolean.

These data types are known as primitive data types. Java also supports user-defined data types that we call object.

In variable declaration, we must specify data type and variable name. Variable name provides reference to the memory location, and data type specifies the type of data this variable will be used to store.

Data type also specify what types of operations that can be performed on those variables. Differences between Numeric data type are range and storage size. Numeric data types can be used arithmetic operation.

doubledecimal number with up to 15 decimal digits

Constant

Like a variable, a constant used to store data. Constant value do not change during program execution. Syntax for constant declaration.

final dataType CONSTANTNAME = value;

When a constant is declared, it must be initialized with a initial value.

final double INTEREST_RATE = 3.5;

By convention, a constant name is written is uppercase. Using a constant make programming easy to maintain, we only need to change the value at one location. A descriptive name for a constant makes program easy to read.
