Operators

Operators are used in operations such as arithmetic, comparisons and logical operations. Operands are the component in an operation that the operator will perform some work on. All operators are typically binary because they have to operands on left side and right side.

operand1 operator operands2

For example, consider expression 3 + 4, the value 3 and 4 are the operands and the operator is addition (+).

Assignment operator

Assignment operation is used to assign a value to a variable. Variables are same as mathematical algebra where its value can change. Assignment operator is the equal (=) sign.

variable = value;

Value can come from a literal value, or an expression like arithmetic operations. Value is written at the right side of the operator and variable on the right.

a = 3;
          
a = 2 + b;

Shorthand operators

Shorthand operators to reuse same variable in a statement. The same variable is used in some calculation where its current value is used, modified, and then reassigned with modified value. For example:

count = count + 10;

In the example above, current count value added with 10 and the result is assigned back to variable count. This expression can be rewritten using shorthand operator as combination of addition and assignment operator as below:

count += 10;

Increment and decrement

Some operators are unary, only one operand in expression, either on left side or right side of the operator.

or

for example, consider increment operation ++a, where variable a is the operand and the operator is increment (++).

int a = 3;
          
System.out.println(a++); // display 4

Conditional operator

There is only one ternary operator in Java. It has two operator and three operands.

the conditional operator is consist of ? and : symbols. For example:

int ans = (a > 5) ? 1.5 : 0.8;
            
System.out.println(ans);// display 0.8
