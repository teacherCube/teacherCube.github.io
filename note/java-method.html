Method

Method is a group of statement that perform certain action. To execute actions stated in a method, the method is called. Method is an example of code reuse. Once method is created, it can be used many times.

Method declaration

Method is declared inside a class

class MyApp{
  public static void main(String[] args){
  
  }

  // method 1 declaration

  // method 2 declaration

  // method 3 declaration

}

Method declaration is written as:

modifier returnType methodName(parameters){
  // method body
}
methodName
- the name of the method
parameters
- the information passed to the method. It is written inside a parentheses. Parameter acts as a local variable that is used inside the method. Multiple parameters can be passed by seperating them by comma.
returnType
- the data type of the value returned by the method. It can be int, float char,etc. If the method method return a value, it should have a return statement. If the method do not return value, void keyword is used.
modifier
- keyword such as static is used specify the method is belong to main class, and not belonging to an object. Other keyword such as public or private indicates accessibility of the method.
public static void sayHello(){
  System.out.println("Hello World");
}

Or,

public static int add(int num1, int num2){
  int total;
  total = num1 + num2;
  return total;
}

Method Call

To use a method, the method need to be called.

public static void main(String[] args){
  sayHello();
}

calling a method with a parameters

public static void main(String[] args){
  int answer = add(3, 5);
  System.out.println(answer);
}

In above example, the value 3 and 4 are called argument. Arguments are values passed to the parameters in a method. THe value returned by the method is assigned to variable called answer in main method.
