Repetition

Loop or repetition statements are to repeat statement execution until stopped. There are three types of repetition statements: for, while and do..while

While statement

Using while loop, if the condition is true, the statements are executed again.if the condition is false the loop is exited or terminated.

while(condition){
  // statement(s)
}

The statement may not execute at all if the condition is false when executing the loop statement for first time. To execute statement at least once, we can use do...while loop.

For example, display 1 to 5 using while loop.

int a = 1;
while(a <= 5){
  System.out.println(a);
  a++;
}

For statement

The for loop is used when as already know how many time to loop. It has three parameters, init, condition and update. These parameters are seperated using semicolon. 

for(init; condition; update){
  // statement(s)
}
init
Initializes the variable used to control the the loop.
condition
Checks control variable and decides to loop again or stop. Condition is written in boolean expression.
update
Update the control variable after each loop.

For example, display 1 until 5 using the for loop.

for(int a = 1; a <= 5; a++){
  System.out.println(a);
}

Break and Continue

the break statement is used to terminate a loop before stopping condition is met.

int a = 1;
while(a <= 5){
  System.out.println(a);
  a++;
  if(a == 3)
    break;
}

the continue statement is used to skip one repetition in the loop and skips to next repetition.

int a = 1;
while(a <= 5){
 
  if(a == 3){
    a++;
    continue;
  }
  
  System.out.println(a);
  a++;
}
