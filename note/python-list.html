List

A post by Cuber

A python list is used to store a sequence of data called elements. Each element is assigned with an integer known as an index. Index values start from zero, and increment by 1.

Access elements

Uses index value to access specific element in list. starts from zero 0

listName[index]

Example

myList = [4,5,2,6]

element1 = myList[0]
element2 = myList[1]
element3 = myList[2]
element4 = myList[3]

print("First element: ", element1)
print("Third element: ", element3)
First element:  4
Third element:  2

negative values allow element access from behind. -1 is last element, -2 second last element and so on...

myList = [4,5,2,6]

element1 = myList[-4]
element2 = myList[-3]
element3 = myList[-2]
element4 = myList[-1]

print("First element: ", element2)
print("Third element: ", element4)
First element:  5
Third element:  6

Modify

Index value also used to change or modify the value of an element.

listName[index] = newValue

Example

myList = [4,5,2,6]
print("Original: ", myList)
myList[2] = 888
print("After modify: ", myList)
Original:  [4, 5, 2, 6]
After modify:  [4, 5, 888, 6]
>

If the index is out of range following error will occur

IndexError: list assignment index out of range

Extend

The extend() function iterates over an iterable object like a list or tuple and adds each item to the list that invokes the extend function.

animals=['dog','cat']
newAnimals=['bird','zebra']

animals.extend(newAnimals)
print(animals)

Output

['dog', 'cat', 'bird', 'zebra']
>

Besides the append() function, another list can be appended using the plus (+) sign to append the list.

animals = ['dog','cat']

animals = animals + ['bird','hamster']
print(animals)

Output

['dog', 'cat', 'bird', 'hamster']
>

If a string is passed to the extend() function, each element is iterated and added one by one as the list item.

animals = ['dog','cat']

animals.extend('bird')
print(animals)

Output

['dog', 'cat', 'b', 'i', 'r', 'd']
>
