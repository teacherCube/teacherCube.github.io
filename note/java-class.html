Class

Class is the blueprint for creating objects. Objects are an instance of a class.

Defining class

class ClassName{
    // fields
    // methods        
}

For example, following class defines a Circle class. It has two fields radius and color and one method called calculateArea()

class Circle{
    double radius;
    String color;

    double calculateArea(){
        double area;
        area = Math.PI * Math.pow(this.radius, 2);
        return area;
    }       
}

This class should be saved in a file called Circle.java in the project src folder.

Create object

Object is created using the new keyword.

ClassName objectReference = new ClassName();
class App{
    public static void main(String[] args){
        Circle circle1 = new Circle();
    }
}

Accessing object properties

Object properties accessed using dot operator. Fields are accessed as follows:

objectReference.fieldName;

and, methods are accessed as below

objectReference.methodName(parameters);
class App{
    public static void main(String[] args){
        Circle circle1 = new Circle();
        circle1.radius = 3.5;
        double area = circle1.getArea();
        System.out.println("area " + area); 
    }
}

Constructor

Constructors creates object and initializes them. Constructors are called by the new keyword, when object is created.

ClassName(){
    fieldName = initialvalue;
}

Example

class Circle{
    double radius;

    Circle(){
      radius = 1.0;
    }

    double getArea(){
      double area;
      area = Math.PI * Math.pow(this.radius, 2);
      return area;
    }
}
