Array

Array is a collection values stored as a single name. Array elements are stored continously in a memory. The elements stored in an array must have same data type. The element are indexed.

Array Declaration

array need to be declared before they can be used

dataType[] arrayName;

Create an Array

Declaration just creates reference, not the array itself. To create an array and assign to array reference:

arrayName = new dataType[size];

the new keyword, creates an empty array of mentioned data type with the size given in the square bracket.

For example, create an empty integer array with size of 5. Then assign it to the array variable nums.

int[] nums;
nums = new int[5];

Or, declare and create array in one statement

int[] nums = new int[5];

Initialize array

An array created is initialized with default value. Default value for integer array is 0. We can initialize while creating array.

dataType[] arrayName = {item1, item2, item3};

The initial values are written in curly bracket, seperated by comman. Array size depends of number of items in bracket.

For example

int[] nums = {2,4,1,2,5,3};

float[] points = {2.1, 3.3, 4.2};

char[] letters = {'f','g', 'h','i'};

String[] words = {"xyz","abc","fgh"};

Access array element

To retrieve the value of element from a specific index number.

arrayName[index];

The index is written in the square bracket. Index start from zero (0), so index for element-n is n - 1.

Example

int[] nums = {2,4,1,2,5,3};

System.out.println(nums[3]);// display 2

int a = nums[2] + num[4];// a = 1 + 5

System.out.println(a);// display 6

Update array element

To assign a value to an element. Index is used to specify which element to assign the new value

arrayName[index] = value;

Specify which element and use the assign operator to update the value.

Example

int[] nums = {2,4,1,2,5,3};

nums[3] = 88;

System.out.println(nums[3]);// display 88

Get array length

To get how many elements are there in an array. This is most commonly used information about an array. The lenght of array is obtained by using the length field of the array object.

int len = arrayName.length;

Example

int[] nums = {2,4,1,2,5,3};

System.out.println(nums.length);// display 5
