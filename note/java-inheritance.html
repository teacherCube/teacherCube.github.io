Inheritance

Inheritance is deriving a new class from an existing class. It inherits properties of its parent class.

Defining inheritance

Inheritance id defined using the extends keyword

class ChildClass extends ParentClass{
    // fields and methods of child class
}

Example

class Shape{
    double width;
    double height;
    String color;

    void displayColor(){
        System.out.println(color);
    }
}

now, Circle class inherits the Shape class. 

class Circle extends Shape{
    double radius;

    void calculateCircleArea(){
        double area = Math.PI * this.radius * this.radius;
        System.out.println("Area of circle is " + area);
    }
}

Circle is the child class and Shape is parent class. By doing this Circle inherits the name, color and count attributes from Shape. Inheritance also known as is-a relationship, so Circle is-a Shape. Circle inherits all the attributes and behaviors of the Shape. 

class App{
    public static void main(String[] args){
        Circle cir = new Circle();
        cir.color = "red";
        cir.displayColor();
    }
}

Output

red

Inheritance types

Single inheritance

Single inheritance is when single child class is inherited from single parent class.

Hierachical inheritance

when there is more than one child class is derived from single parent class.

Multilevel inheritance

when a class derived from another class which also derived from another class.

Multiple inheritance

when a child class is derived from two or more parent class.

*Java do not support this type of inheritance. Can be achived using interface.

Hybrid inheritance

Combination of two of more types of inheritance discussed above. In java, all inheritance type except the multiple inheritance.
