Decision

Selection statement are also known as if statements are used to make decisions during program execution. It uses conditions, expressed using Boolean expression to help make this decisions.

If statement

If the condition is true than execute the statement, otherwise don't execute the statement.

if(condition){
    // statement(s)
}
int a = 5;

if(a == 5){
    int temp = a + 2;
    System.out.println(temp);// displays 7
}

If...else statement

If the condition is true than execute the first statement, otherwise execute the second statement.

if(condition){
    // first statement(s)
}else{
    // second statement(s)
}
int a = 8;

if(a == 5){
    int temp = a + 2;
    System.out.println(temp);
}else<{
    System.out.println(a); // displays 8
}

If...else if statement

If the condition one is true than execute the first statement. If condition 1 is false, check condition 2. If condition 2 is true than execute the second statement, otherwise execute the last statement.

if(condition1){
    // first statement(s)
}else if (condition2){
    // second statement(s)
}else{
    // last statement(s)
}

We may specify as many else if blocks as needed to solve a problem.

int a = 8;

if(a == 1){
    System.out.println("One");
}else< if(a == 2){
    System.out.println("Two");
}else< if(a == 3){
    System.out.println("Three");
}else{
    System.out.println("Unknown");
}

Switch

switch (expression){
    case value: 
          // statement(s)
          break;
    case value: 
          // statement(s)
          break;
    default: 
          // statement(s)
}

Expression is evaluated once to obtain its value. The value is checked with the case value to find a match. statement corresponding to matching value is executed. If no match found, execute statement specified as default. The break and default are optional. Break stops execution from executing the statements in following case.

int a = 2;

switch (a){
    case 1: 
        System.out.println("One");
        break;
    case 2: 
        System.out.println("Two");
        break;
    case 3: 
        double temp = a * 1.5;
        System.out.println(temp);
        break;
    default: 
        System.out.println("Unknown");
}
